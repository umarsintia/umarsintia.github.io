Cara Membuat Akun GitHub Baru

  1. Buka Browser Web:

    2. Buka browser seperti Google Chrome, Mozilla Firefox, atau Microsoft Edge di komputer Windows anda.

  2. Kunjungi Situs Web GitHub:

    3. Ketikkan "github.com" dalam bilah alamat browser anda dan tekan Enter. Ini akan membawa Anda ke situs web resmi GitHub.

  3. Pilih "Sign Up" atau Sign up for GitHub":

    4. Di halaman depan GitHub, Anda akan melihat tombol "Sign Up" atau "Sign up for GitHub". Klik tombol ini.

  4. Isikan informasi pendaftaran:

    5. Anda akan diarahkan ke halaman pendaftaran. Di sini, Anda harus mengisi beberapa informasi dasar seperti:

    • Nama pengguna (Username): Pilihlah nama pengguna yang unik yang akan digunakan untuk akun GitHub Anda.
    • Alamat Email: Masukkan alamat email yang aktif dan dapat diakses.
    • Kata Sandi(Password): Buat Kata sandi yang kuat dan aman untuk melindungi akun anda.

  5. Verifikasi akun anda:

    6. Setelah mengisi informasi tersebut, Anda akan diminta untuk mengisi verifikasi CAPTCHA atau tanda bahwa anda bukan robot.

  6. Pilih Jenis Akun:

    7. Anda akan diberikan pilihan antara "Free" (Gratis) dan "Team" (Tim). Pilih "Free" jika Anda ingin membuat akun GitHub gratis untuk penggunaan pribadi. Jika Anda ingin membuat akun untuk tim dan organisasi, pilih "Team" dan ikuti langkah-langkah yang diberikan.

  7. Konfigurasi profil:

    8. Setelah mengaktifkan akun, Anda dapat mengonfigurasi profil GitHub dengan informasi tambahan seperti foto profil dan deskripsi singkat.

Cara Mengupload Pertama Kali di GitHub

  1. Membuat Repositori di GitHub:
    • Masuk ke akun GitHub Anda di browser web.
    • Klik tombol "New" atau "New repository" di dashboard atau halaman utama GitHub Anda.
  2. Isi Informasi Repositori:
    • Berikan nama untuk repositori Anda. Ini akan menjadi nama folder di GitHub.
    • Anda dapat menyertakan deskripsi singkat (opsional) untuk menjelaskan tujuan repositori.
    • Pilih apakah repositori ini akan bersifat publik (visible to everyone) atau privat (hanya Anda dan kolaborator Anda yang dapat melihatnya).
    • Jika Anda inginmenginisialisasi repositori dengan file README, Anda dapat memilih opsi "Initialize this repository with a README". Ini biasanya berguna untuk memulai proyek.
  3. Tambahkan File ke Lokal Repositori:
    • Untuk mengunggah file pertama kali, Anda perlu memilikinya di komputer Anda. Pastikan Anda memiliki file atau proyek yang ingin diunggah.
    • Buat atau pindahkan file tersebut ke direktori lokal yang digunakan untuk repositori Anda.
  4. Inisialisasi Git di Direktori Lokal:
    • Buka terminal atau command promt di komputer Anda
    • Navigasikan ke direktori tempat Anda menyimpan file untuk repositori Anda menggunakan perintah 'cd' (contoh: 'cd /path/to/your/repository').
  5. Inisialisasi Git dalam Repositori Lokal:
    • Jalankan perintah 'git init' untuk menginisialisasi Git dalam direktori lokal tersebut.
  6. Tambahkan File ke Staging Area:
    • Gunakan perintah 'git add .' untuk menambahkan semua perubahan file ke dalam staging area. Jika Anda hanya ingin menambahkan file tertentu, gantilah titik dengan nama file yang Anda inginkan.
  7. Commit Perubahan:
    • Jalankan perintah 'git commit -m "Pesan Commit Anda"' untuk membuat commit lokal dengan pesan yang menjelaskan perubahan yang Anda buat.
  8. Tautkan repositori lokal dengan repositori GitHub Anda:
    • Kembali ke halaman repositori GitHub anda di browser web.
    • Salin URL repositori yang terlihat dibagian atas (biasanya berbentuk 'https://github.com/username/repository.git').
  9. Tambahkan Remote Repository:
    • Kembali ke terminal atau command prompt
    • Jalankan perintah 'git remote add origin https://github.com/username/repository.git' (gantilah URL dengan URL yang Anda salin).
  10. Push ke GitHub:
    • Terakhir, jalankan perintah 'git push -u origin master' (jika Anda menggunakan cabang master). Ini akan menggunakan commit Anda ke repositori GitHub Anda.
